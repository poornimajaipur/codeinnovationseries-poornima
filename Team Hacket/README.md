# Collab-O-Mate 
by **Team Hacket**</br>
This is a Project Collaboration platform for college students where ideators can meet innovators. </br>
A person can create an open project idea and that idea will be available to thousands of students of that college and other people who like this idea can collaborate with it to help make the idea come to life. 

### Features
- Open Ideas for everyone 
- Account login and verification
- Comment Your Suggestions for betterment of Ideas
- Ask to be a Collaborator to help build the project
- **Hall of Fame** for Successfull Ideas.

### Installation
- Clone the repo and `cd` into it
- Run `pip install -r requirements.txt` and install all dependencies
- Change the `db.yaml` file and set `mysql_password` as your MySql password. **Very Important**
- In MySQL Client, Run `SOURCE collabomate_dump.sql` to dump MySql database
- Run `python app.py` to start the main server
- Open `http://localhost:5000` on the browser
